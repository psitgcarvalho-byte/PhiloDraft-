Decidir raramente é escolher entre alternativas claras. Na maior parte do tempo, decidir é **interromper** um fluxo: um hábito, uma emoção, uma urgência, uma expectativa do outro.

## Três camadas práticas

1. **A camada afetiva**: o corpo “puxa” para uma direção (aproximação/evitação).
2. **A camada habitual**: o repertório automatiza a resposta.
3. **A camada justificativa**: a linguagem produz uma explicação aceitável.

O ponto-chave é que a justificativa costuma ser a última a chegar — mas é a primeira a ser contada.

## O erro comum

Confundir a explicação com a causa. A pessoa diz “eu fiz porque…” e toma a frase como origem do ato. Muitas vezes ela é apenas o acabamento social do acontecimento psíquico.

## Uma pista clínica

Quando a justificativa muda a cada sessão, isso não prova mentira. Pode provar que a narrativa está tentando acompanhar algo mais profundo: um conflito, uma culpa, uma ambivalência.

A decisão, no fim, é o encontro entre forças internas e restrições externas. A linguagem vem depois para tornar o encontro comunicável.
